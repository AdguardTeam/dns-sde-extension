This extension demonstrates how Structured DNS errors can work if supported natively by browsers.

You should first configure your browser to use AdGuard DNS:

  1. Open Security settings in a new tab: chrome://settings/security
  2. Enable Use secure DNS
  3. In Select DNS provider choose Add custom DNS service provider
  4. Enter https://dns.adguard-dns.com/dns-query

Now every time you try to open a website blocked by AdGuard DNS, the extension will detect that and show you a page indicating that the page was blocked by DNS. To see an example try visiting ad.doubleclick.net.
